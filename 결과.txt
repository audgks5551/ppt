# 작품 설명

 - 구성요소 목록
    1. 피에조 부조
    2. IR센서, 리모콘
    3. 7세그먼트 디스플레이

 - 작동 순서
    1. 7세그먼트 디스플레이에 0번이 표시가 되어 작동 준비가 되어있다는 표시입니다.
    2. 리모콘 1, 2, 3번중 하나를 선택하여 누릅니다.
    3. 7세그먼트 디스플레이가 누른 번호대로 1, 2, 3을 출력합니다.
    4. 1, 2, 3번의 노래가 각각 출력됩니다.
    5. 노래가 재생되는 도중에 초록색 정지 버튼을 누르게 되면 노래가 정지되고 7세그먼트 디스플레이에는 0번이 표시됩니다.
    6. 이 과정을 계속해서 반복합니다.

# 아쉬운 점

 - 노래가 재생이 되었을 때 정지 버튼을 누르고 다시 정지 버튼을 누르면 정지되었던 부분부터 재생이 되어야하는데 다시 처음으로 돌아가서 번호를 선택해야 하는게 아쉽습니다.
